### 五、项目介绍
项目编号：python110D

基于Python+Django的少儿图书推荐系统，为少儿用户提供个性化图书推荐服务，方便管理员进行图书管理和推荐维护，提升少儿阅读体验。

角色：管理员、用户

用户功能：注册登录、浏览图书、搜索查询、图书推荐、图书收藏、个人中心。

管理员功能：登录、用户管理、图书管理、图书分类管理、推荐管理、评论管理、系统管理。

数据库表：14个
